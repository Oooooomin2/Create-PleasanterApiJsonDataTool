## プリザンターAPI Json製造機

本アプリはオープンソースである国産Webデータベース「プリザンター」のAPIに使用するJsonモデルを生成するツールです。

## 使い方

### 下記リンクよりダウンロードし、.exeファイルをクリックすると以下のような画面が表示されます。  

https://github.com/Oooooomin2/Create-PleasanterApiJsonDataTool/releases/download/v1.0/CreatePleasanterApiJsonDataTool.zip

![images](https://user-images.githubusercontent.com/63548353/101233184-21b27700-36fa-11eb-97d0-44adb7bd47f8.png)


### ApiVersion, ApiKeyを入力し、「Jsonを生成する」ボタンをクリックするとJsonが出力されます。

![images2](https://user-images.githubusercontent.com/63548353/101233214-69390300-36fa-11eb-9b51-71bc2426bfa0.png)

## 制約

本アプリではテーブルのレコードにおける取得/作成/更新/削除のJsonしか製造しない仕様となっております。
後々は他APIのJson等も作成できるよう機能追加していくかもしれません。
